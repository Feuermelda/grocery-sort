# Grocery Sort

Grocery Sort is a collection of projects focused on organizing grocery lists based on store layouts and shopping flow.

## Projects
- `wpf-grocery-sort`: WPF desktop application with SQLite, custom store layouts, favorites, CRUD functionality, and automatic item sorting
- `grocery_sort.py`: Original Python prototype using JSON-based storage and store-specific sorting logic

## Goal

The goal of this repository is to explore practical sorting logic for everyday grocery shopping and turn it into usable applications.
